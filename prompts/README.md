## Why Prompts?

While dossiers automate **project operations** (setup, tasks, worktrees), prompts will help with **code operations** (writing, reviewing, refactoring code).

**Example**:
```
Use MI6 code-review prompt to review this PR:
- Check for security vulnerabilities
- Validate error handling
- Verify test coverage
- Suggest improvements
```
